## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.comh/victor-vas/covid-19-pwa covid-19-pwa

# Go into the repository
$ cd covid-19-pwa

# Install dependencies
$ yarn

# Run the app
$ yarn start
```
